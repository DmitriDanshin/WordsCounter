# Запуск

1. Распаковать архив.
2. Зайти в папку WordsCounter-master/venv/Scripts.
3. Запустить через консоль.
   
   Windows:
   
        activate.bat
   Unix или MacOS:
   
        bin/activate

4. В WordsCounter-master введите:
    
        python main.py

5. Введите URL адресс в формате.
   
        https://www.python.org/
